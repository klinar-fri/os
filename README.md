## Domača naloga pri predmetu Operacijski sistemi (mini lupina).
- Implemetacija svoje mini lupine: mysh
    - Lupina vsebuje kar nekaj vgrajenih ukazov (več kot 30).
    - Ima tudi podporo za izvajanje zunanjih ukazov (Ukazov iz Linux okolja).
    - Omogoča grajenje ukaznih cevovodov, preusmerjanje vhoda in izhoda<br>
    ter izvajanje v ozadju.
    - Z ukazom $help$ izpišemo pomoč in sezam ukazov, njihov način uporabe pa lahko<br>
    enostavno preverimo z kombinacijo $help$ $ukaz$.
    - Lupino enostavno prevedemo z svojim najlubšim c-prevajalnikom in poženemo preveden program.

## Home work for subject Operating systems (mini shell).
- Implemetation of my mini shell: mysh
    - Shell implements many builtin commands (more than 30).
    - It has support for executing external commands (Linux platform specific commands).
    - It supports pipeline building, redirecting input and output and executing commands<br>
     in the backround (subshell).
    - The $help$ command displays the list of builtin commands, their usage can be easly<br>
    checked with the combination of $help$ $command$.
    - We compile the myshell.c progam with our favourite c-compiler and that's it.
